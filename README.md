# NudeNet WebUI

A browser interface for NudeNet classifier.

## Features

- Download and test image from image URL
- Test images in local directory

## Installation & Running

1. Install the dependencies:

```bash
pip install -e .
```

2. Run server:

```bash
python index.py
```

3. Open browser `http://127.0.0.1:5000`
